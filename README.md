# Dropstone CLI Homebrew tap

Official Homebrew tap for [Dropstone CLI](https://dropstone.io).

## Install

```bash
brew tap blankline-org/dropstone-cli
brew install dropstone
```

## Upgrade

```bash
brew upgrade dropstone
```

## Verify

```bash
dropstone --version
```

## What is this?

Dropstone CLI is a coding agent that runs in your terminal. This repository only contains the Homebrew formula; binaries are hosted on a separate CDN.

Learn more at [dropstone.io](https://dropstone.io).
